##  Commands Overview

Some common commands:

* `/play <song>` → Play a song
* `/pause` → Pause the current track
* `/resume` → Resume paused music
* `/skip` → Skip to the next track
* `/stop` → Stop playback
* `/queue` → Show the queue
* `/filter <type>` → Apply filters (bass, nightcore, etc.)
